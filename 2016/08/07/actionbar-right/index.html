Actionbar左侧标题栏添加和实现切换的功能

很多界面的顶部标题会有其他功能，例如切换fragment，弹选择地区或性别等的弹窗，以下为具体的代码实现，当然同样是继承AppCompatActivity。

在Activity的oncreat方法中添加下面的代码：
第一行代码是允许进行自定义布局的显示，第二行代码是注入你自定义的顶部布局。很多时候顶部不仅仅是标题的展示，如果它可以进行点击，照顾到用户的体验性，会在标题胖添加一个小三角什么的。这里的customView就是自定义的标题布局。 

getSupportActionBar().setDisplayShowCustomEnabled(true);
getSupportActionBar().setCustomView(customView);

我的项目中，有需求是需要在顶级顶部标题时，弹出一个含列表的popupwindow，然后点击的时候，切换不同的fragment，并记住用户的此次的选择。记住用户选择就暂用伪代码代替了。定义一个int标量，默认值为0，也就是进入界面后，默认加载是列表中的第一项。在setAdapter的getView方法时进行判断。根据上次记住的lastChoose 变量值，来确定弹窗弹出后哪一条进行选定或者进行标示。

private int lastChoose = 0;
customView.setOnClickListener(new OnClickListener() {

if (position == lastChoose) {
//变蓝和显示小三角
viewHolder.state.setTextColor(getResources().getColor(
                                    R.color.blue));
                            viewHolder.choose.setVisibility(View.VISIBLE);
                        } else {
//变黑和隐藏小三角
viewHolder.state.setTextColor(getResources()
                                    .getColor(R.color.black));
viewHolder.choose.setVisibility(View.GONE);
                        }
});

与上次所选的项目不同时，才进行数据的加载或者刷新 

if (lastChoose != position) {
        changePage(position);
}
